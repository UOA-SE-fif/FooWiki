# FooWiki

本项目预计使用Flask做为后端框架，bootstrap5作为前端框架

### 项目结构
```
└── application
    ├── config           #配置文件
    ├── core             #核心算法
    ├── models           #模型（与数据库设计相对应）
    ├── routes           #路由
    ├── static           #前端静态文件夹
    │   └── bootstrap
    └── templates        #Flask模版文件夹
